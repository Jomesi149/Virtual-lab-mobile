# Mobile App Assets

Folder ini berisi asset untuk aplikasi mobile.

## File yang Diperlukan

Untuk menjalankan aplikasi dengan benar, tambahkan file-file berikut:

### 1. icon.png
- Ukuran: 1024x1024 px
- Format: PNG
- Deskripsi: Ikon aplikasi utama

### 2. splash.png
- Ukuran: 1284x2778 px (atau sesuai kebutuhan)
- Format: PNG
- Deskripsi: Splash screen saat aplikasi dibuka

### 3. adaptive-icon.png
- Ukuran: 1024x1024 px
- Format: PNG
- Deskripsi: Ikon adaptif untuk Android

### 4. favicon.png
- Ukuran: 48x48 px
- Format: PNG
- Deskripsi: Favicon untuk versi web

## Membuat Icon dengan Expo

Anda juga bisa menggunakan tool online untuk membuat icon:
- https://easyappicon.com/
- https://appicon.co/
- https://makeappicon.com/

Atau menggunakan Figma/Adobe XD untuk desain custom.
